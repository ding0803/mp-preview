## 安装步骤

### 方法一：手动复制（推荐）

1. **找到你的 Obsidian vault 目录**
   - 例如：`D:\Obsidian\我的笔记`

2. **创建插件目录**
   ```
   你的Vault目录/.obsidian/plugins/mp-preview/
   ```

3. **复制以下文件到插件目录**
   - `main.js`
   - `manifest.json`

4. **在 Obsidian 中启用插件**
   - 打开 Obsidian
   - 进入 设置 → 社区插件
   - 关闭"安全模式"（如果开启）
   - 在已安装插件列表中找到"微信公众号预览"
   - 点击启用

### 方法二：使用命令一键安装

在命令行中执行（将路径替换为你的 vault 路径）：

```powershell
# 替换 $VAULT_PATH 为你的 vault 路径
$VAULT_PATH = "D:\你的ObsidianVault路径"
$PLUGIN_DIR = "$VAULT_PATH\.obsidian\plugins\mp-preview"

# 创建插件目录
New-Item -ItemType Directory -Force -Path $PLUGIN_DIR

# 复制文件
Copy-Item "main.js" -Destination $PLUGIN_DIR
Copy-Item "manifest.json" -Destination $PLUGIN_DIR

Write-Host "安装完成！请在 Obsidian 中启用插件。"
```

## 使用方法

1. 在 Obsidian 中打开 Markdown 文件
2. 点击右侧图标打开公众号预览面板
3. 在设置中选择你喜欢的主题
4. 点击"复制到公众号"或"发布到公众号"

## 常见问题

**Q: 插件没有显示？**
A: 请检查是否关闭了"安全模式"，设置→社区插件→关闭安全模式

**Q: 构建失败？**
A: 运行 `npm install` 安装依赖后再执行 `npm run build`
